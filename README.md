# LLM E-commerce Chatbot

Бета-версия нтеллектуальный чат-бота для электронной коммерции, построенный на основе LangChain и интегрированный с OpenRouter API для обработки естественного языка. Будет производиться дальнейшее улучшение данной системы.

## Возможности

- **Умная обработка запросов**: Использование современных языковых моделей для понимания и ответов на вопросы клиентов
- **FAQ система**: Автоматический поиск ответов в базе часто задаваемых вопросов
- **Отслеживание заказов**: Проверка статуса заказов по номеру через команду `/order`
- **История диалога**: Сохранение контекста разговора для более естественного общения
- **Логирование**: Ведение журнала всех сессий для анализа и отладки
- **Русскоязычный интерфейс**: Полностью локализованный для русскоязычных пользователей

## Требования

- Python 3.8+
- API ключ от OpenRouter

## ️ Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/ThomasShelby228666/LLMApp.git
cd LLMApplication
```
2. Создайте виртуальное окружение:
```bash
python -m venv .venv
source .venv/bin/activate  # для Linux/Mac
.venv\Scripts\activate     # для Windows
```
3. Установите зависимости:
```bash
pip install -r requirments.txt
```

4. Настройте переменные окружения (создайте файл .env в корне проекта):
```bash
OPENROUTER_API_KEY=your_api_key_here
OPENROUTER_MODEL_NAME=your_model_name
```

Запуск основного бота
```bash

```

## Доступные команды
- Общение: Просто задайте вопрос боту на русском языке

- Проверка заказа: ```/order 12345``` (где 12345 - номер заказа)

- Очистка контекста: ```сброс``` - очищает историю диалога

- Выход: ```выход, стоп или конец``` - завершение сессии
